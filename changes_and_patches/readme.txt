This folder will store the patches developed by decathlon.

If the patch applies to openbravo core it should be placed into core folder.
Example:
  I've a patch for openbravo core:
    1. Save it inside core folder

If the patch applies to  specific module it will be stored in a folder. The name of this folder will be the module affected by the patch.
Example:
  I've a patch for "org.openbravo.mobile.core":
    1. create folder (if it doesn't exists)
    2. save patch inside the folder
